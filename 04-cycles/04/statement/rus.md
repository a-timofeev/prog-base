# Сумма цифр

Дано целое число.
Найдите сумму его цифр.

## Формат входных данных

Первая строка содержит целое число $n$ ($0 \leqslant n \leqslant 10^9$).

## Формат результата

Выведите сумму цифр числа $n$.

## Примеры

------------------------------
входные данные  результат
--------------  --------------
0               0

231             6
------------------------------
