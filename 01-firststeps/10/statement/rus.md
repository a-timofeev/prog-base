# Модуль числа

Напишите программу, которая выводит модуль числа (функция `abs`).

## Формат входных данных

Дано число $n$ ($-10^6 \leqslant n \leqslant 10^6$).

## Формат результата

Выведите модуль числа.

## Примеры

------------------------------
входные данные  результат
--------------  --------------
-10             10

54              54

0               0
------------------------------
