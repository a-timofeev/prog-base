# Степень двойки

Для заданного целого неотрицательного числа $n$, вычислите $2^n$.
 
## Формат входных данных

Первая строка содержит целое число $n$ ($0 \leqslant n \leqslant 20$).

## Формат результата

Выведите $2^n$.

## Примеры

------------------------------
входные данные  результат
--------------  --------------
0               1

5               32
------------------------------
